**1. Create `src/lib/server/new-account.ts`:**

Export an `isNewAccount(supabase, userId)` async function that returns `boolean`. The function checks two conditions (either triggers "new account"):

- Account age < 7 days: Use the service-role Supabase client (`supabase`) to call `supabase.auth.admin.getUserById(userId)`. Extract `created_at`. If `Date.now() - new Date(created_at).getTime() < 7 * 24 * 60 * 60 * 1000`, return true.
- Fewer than 3 approved ads: Query `supabase.from('ads').select('id', { count: 'exact', head: true }).eq('user_id', userId).eq('status', 'active')`. If `count < 3`, return true.

Wrap both checks in try/catch. On error, log `console.warn('new_account_check_failed', { userId, error: err.message })` and return false (fail-open -- don't penalise users if the check fails).

Import `SupabaseClient` type from `@supabase/supabase-js`. Keep the function pure -- no SvelteKit-specific imports.

**2. Create `src/lib/server/reseller-detection.ts`:**

Export types and function:

```typescript
export type ResellerSignal = {
	signal: string;
	weight: number;
	matched: string;
};

export function detectResellerSignals(title: string, description: string): ResellerSignal[];
```

Define `DEALER_PATTERNS` array of objects `{ pattern: RegExp, signal: string, weight: number }`:

- `/\bcall\s+for\s+price\b/i` -- weight 2 (strong signal)
- `/\bstock\s*#?\s*\d/i` -- weight 2
- `/\bRRP\b/i` -- weight 1
- `/\btrade[\s-]*in/i` -- weight 1
- `/\bfinance\s+available\b/i` -- weight 2
- `/\bshowroom\b/i` -- weight 2
- `/\bwarranty\s+included\b/i` -- weight 1
- `/www\.\w+\.\w+/i` -- weight 1
- `/https?:\/\/\S+/i` -- weight 1
- `/\b(?:0\d{1,2}[\s.-]?\d{6,8}|\+\d{1,3}[\s.-]?\d{6,12})\b/` -- weight 1 (phone number)
- `/\bprice\s+list\b/i` -- weight 2
- `/\binstallment/i` -- weight 1
- `/\bdelivery\s+available\b/i` -- weight 1

Concatenate title + " " + description, iterate patterns, collect matches. Return array of matched signals.

Also export `const RESELLER_THRESHOLD = 2` -- the minimum total weight to flag.
